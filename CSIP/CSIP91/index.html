CSIP91

Testable: TRUE
Schema Valid: Valid
Description: If at least one administrative metadata unit (i.e. a techMD, rightsMD, sourceMD or digiprovMD element under mets/amdSec) exists then mets/structMap[@LABEL='CSIP']/div/div[@LABEL='Metadata']/@ADMID MUST exist and refer to them by their @ID as a space-separated list of @ID values (the attribute is of type xsd:IDREFS).

Rules

  • Rule ID: 1

    Descripton: If there are amdSec IDs then the @ADMID attribute MUST exist

    ERROR: There are administrative metadata described in amdSec and therefore there must be a mets/structMap[@LABEL='CSIP']/div/div[@LABEL='Metadata']/@ADMID. All amdSec ID's MUST be listed in the @ADMID using spaces as delimiters.
